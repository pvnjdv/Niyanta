TASK 2.2 — In-memory data stores
  Declare these at the top of server.js after imports:

  const auditStore = []         // max 500 entries, newest first
  const sessionStore = new Map()  // sessionId -> session object
  const metricsStore = {
    totalWorkflowsRun: 0,
    totalTasksCreated: 0,
    totalDecisionsMade: 0,
    escalationsTriggered: 0,
    processingTimes: [],         // array of ms values for avg calculation
    agentRunCounts: {
      meeting: 0, invoice: 0, hr: 0, procurement: 0, security: 0
    },
    decisionBreakdown: {
      approved: 0, flagged: 0, rejected: 0,
      critical: 0, proceeded: 0, escalated: 0
    },
    serverStartTime: Date.now()
  }

TASK 2.3 — Agent system prompts object
  Define AGENT_PROMPTS object in server.js with all 5 agent system prompts.
  Each prompt instructs the model to return ONLY valid JSON.
  See REQUIREMENTS.md Section 5 for full prompt text for each agent.

TASK 2.4 — Helper functions
  Write these helper functions:

  function addAuditEntry(agentId, event, decision, inputPreview, processingTime)
    Creates audit entry object with: id (uuid), agentId, event, decision,
    inputPreview (first 80 chars), processingTime, timestamp (ISO string)
    Unshifts to auditStore. If auditStore.length > 500, pops last item.

  function updateMetrics(agentId, result, processingTimeMs)
    Increments totalWorkflowsRun.
    Adds processingTimeMs to processingTimes array.
    Increments agentRunCounts[agentId].
    Counts tasks/checklist/next_steps from result and adds to totalTasksCreated.
    If result.decision exists: increments totalDecisionsMade,
      updates decisionBreakdown based on decision value.
    If result.severity === CRITICAL or result.escalate_to_human === true:
      increments escalationsTriggered.

  function parseAgentResponse(text)
    Strips markdown code fences (```json, ```)
    Trims whitespace
    Attempts JSON.parse
    On failure: returns { summary: text, audit: "Raw response - JSON parse failed" }

TASK 2.5 — Middleware setup
  app.use(helmet())
  app.use(cors({ origin: ["http://localhost:3000", "http://localhost:3001"] }))
  app.use(express.json({ limit: "2mb" }))
  app.use(morgan("dev"))

  Rate limiter for agent routes:
  const agentLimiter = rateLimit({
    windowMs: 60000,
    max: 60,
    message: { error: "Too many requests", message: "Rate limit exceeded. Wait 1 minute." }
  })
  Apply agentLimiter to /api/agent/* and /api/niyanta/*

TASK 2.6 — POST /api/agent/run endpoint
  Validate req.body.agentId is one of the 5 valid agent IDs.
  Validate req.body.input is a non-empty string.
  If validation fails: return 400 with error message.

  Create sessionId = uuid()
  Record startTime = Date.now()

  Add to sessionStore: { agentId, startTime, status: "processing", input: req.body.input }

  Call Anthropic API:
    model: "claude-sonnet-4-20250514"
    max_tokens: 1500
    system: AGENT_PROMPTS[agentId]
    messages: [{ role: "user", content: req.body.input }]

  Record processingTime = Date.now() - startTime
  Parse response with parseAgentResponse()

  Update sessionStore entry: status "complete", processingTime
  Call updateMetrics(agentId, result, processingTime)
  Call addAuditEntry(agentId, result.summary, result.decision || result.severity, ...)

  Return 200 with:
  {
    success: true,
    sessionId,
    agentId,
    result,
    processingTime,
    timestamp: new Date().toISOString()
  }

  Wrap in try/catch. On error: return 500, update session status to "error",
  add error audit entry.

TASK 2.7 — POST /api/niyanta/chat endpoint
  Receive: { message, conversationHistory, agentResults }
  Build context string from agentResults:
    Stringify each result's summary + decision/severity into readable context.

  System prompt:
    "You are Niyanta, the Autonomous Enterprise Governor AI. You are the master
    orchestrator of 5 enterprise agents. You have complete situational awareness.
    You are decisive, authoritative, and concise. You proactively identify
    cross-workflow dependencies. Current agent context: [context string]"

  Build messages array from conversationHistory + new message.

  Call Anthropic API:
    model: "claude-sonnet-4-20250514"
    max_tokens: 1000
    system: niyantaSystemPrompt
    messages: messagesArray

  Return 200 with:
  {
    reply: responseText,
    timestamp: new Date().toISOString()
  }

TASK 2.8 — GET /api/audit endpoint
  Return last 100 entries from auditStore (already sorted newest first).
  Return: { entries: auditStore.slice(0, 100), total: auditStore.length }

TASK 2.9 — GET /api/metrics endpoint
  Calculate avgProcessingTimeMs from metricsStore.processingTimes array.
  Count active sessions (status: "processing") from sessionStore.
  Return full metricsStore object plus:
  - avgProcessingTimeMs (calculated)
  - agentsActive (count of processing sessions)
  - uptimeSeconds (Date.now() - serverStartTime) / 1000

TASK 2.10 — GET /api/health endpoint
  Return: { status: "ok", uptime: process.uptime(), timestamp, version: "1.0.0",
            agentsAvailable: 5, totalProcessed: metricsStore.totalWorkflowsRun }

TASK 2.11 — Global error handler
  app.use((err, req, res, next) => {
    console.error(err.stack)
    res.status(500).json({ error: "Internal server error", message: err.message,
                           timestamp: new Date().toISOString() })
  })

TASK 2.12 — Start server

    "Niyanta AI Server running on port PORT\nAgents: 5 active\nStatus: OPERATIONAL"

  Manages all 5 agent states.
  State shape per agent:
  {
    status: "idle" | "processing" | "complete" | "error",
    messages: [],     // array of message objects { type, content, timestamp }
    result: null,     // parsed JSON result from API
    taskCount: 0,     // extracted task/item count
    lastActivity: null, // ISO timestamp
    processingTime: null // ms
  }

  Export:
  - agentStates (object with all 5 agent states)
  - runAgent(agentId, inputText) — calls api.runAgent, updates state
  - runAllAgents() — runs all 5 sequentially with SAMPLES data
  - addMessage(agentId, message) — adds to agent's messages array
  - resetAgent(agentId) — resets single agent to idle
  - isAnyProcessing — boolean derived state

TASK 4.2 — src/hooks/useAuditLog.js
  Manages local audit log state.
  addEntry(agentId, message, decision) — prepends to log, caps at 200
  clearLog() — empties log
  filterByAgent(agentId) — returns filtered entries
  Returns: { log, addEntry, clearLog, filterByAgent }

TASK 4.3 — src/hooks/useTheme.js
  Manages dark/light theme.
  Persists to localStorage under key "niyanta-theme".
  On mount: reads from localStorage, defaults to "dark".
  On change: sets document.documentElement.setAttribute("data-theme", theme)
  Returns: { theme, toggleTheme, isDark }

TASK 4.4 — src/hooks/useNiyantaChat.js
  Manages Niyanta Command chat state.

  sendMessage(text, agentResults) — calls api.sendNiyantaMessage,

  Returns: { messages, isLoading, sendMessage, clearChat }
